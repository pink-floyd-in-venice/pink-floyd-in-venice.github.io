Metadata Alignment

In order to make data more interoperable, we grouped the different items in three categories and we analysed the different metadata used for each category. In many cases the metadata used allows a sufficient interoperability between different standards referred to items with similar characteristics; in general, we chose to use Dublin Core (dc) as a reference, as it represents one of the most widely used standards.
